#### 前言
毕设项目，从github中参考了[zhengqingqian](https://github.com/zhengqingqian/pet-web)的比赛项目，并优化了各种功能，额外实现了评论、在线聊天、更完整的个人页面。

此仓库为前端代码，后端项目参考 [Gitee 后端代码托管仓库](https://gitee.com/yang-feng-hank/AnimalAdopt)


### 项目结构

* components --- 组件包
* config    --- 配置包
    * router.js --- 路由配置
    * store.js  --- 单一状态数
* page  --- 主页面
